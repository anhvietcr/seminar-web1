Clone
============
```bash
git clone https://github.com/anhvietcr/seminar-web1.git
composer update
```

### Installing via Composer (If Composer not exists)

- Go Home and Install [Composer](http://getcomposer.org).

- Init composer
```bash
# Init Composer
composer init
```

Config
============
- Install Mysql (web1_seminar)
- /config/database.php
- .env
